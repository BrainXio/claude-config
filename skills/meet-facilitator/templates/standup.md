# Template: Standup

**Use when:** Daily coordination, multi-agent sync, parallel work alignment

**Participants:** All roles (5 min each)

**Duration:** 5 min per participant

______________________________________________________________________

## Structure

Each participant answers three questions:

### What I Did

- Completed since last standup
- Commits, decisions, meetings attended

### What I'll Do

- Next 24 hours
- Blockers to unblock

### Blockers

- What's stuck
- Who can help

______________________________________________________________________

## Role-Specific Focus

### Architect

- Design decisions made
- Documentation written
- Implementation progress

### Critic

- Risks identified
- Code reviews completed
- Edge cases documented

### Stakeholder

- User feedback received
- Priority changes
- Business constraint updates

______________________________________________________________________

## Output

```markdown
## Standup: YYYY-MM-DD

| Participant | Did | Doing | Blockers |
|-------------|-----|-------|----------|
| Architect | [summary] | [summary] | [none / help needed from X] |
| Critic | [summary] | [summary] | [none / help needed from X] |
| Stakeholder | [summary] | [summary] | [none / help needed from X] |
```

______________________________________________________________________

## Anti-Patterns

- **Problem-solving** — Don't solve blockers in standup; take it offline
- **Status reporting** — This is coordination, not accountability
- **>5 min per person** — Timebox strictly; continue offline
